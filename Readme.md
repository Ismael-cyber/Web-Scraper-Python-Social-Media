# WebScraper para Rede Social

## Descrição do Projeto

Criar uma ferramente de negócio, capaz de obter dados de perfis na rede social que possam contribuir para uma empresa na qual seus leads (potenciais clientes) estejam nesta plataforma.

## Obstáculo do Problema

A plataforma em si, criou estratégicamente, mecanismos contra web scrapers, dificultando na obtenção e simulação de login, fazendo nosso algoritmo ser muito mais mutável antes mesmo da distribuição da ferramenta, rebaixando nosso projeto sempre, como **BETA**.

## Stacks

Nossa ferramenta utiliza a linguagem **Python** juntamente com as bibliotecas:

- Playwright
- Logging
- Dotenv
- Os
- Time
